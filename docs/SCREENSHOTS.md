## How to Take Screenshots

1. Open the live app in Chrome
2. Switch to English language (EN button, top-right)
3. Disable browser extensions
4. Set zoom to 100%
5. Use Chrome DevTools > Device toolbar for exact dimensions
6. Save as PNG to `docs/` folder

## File Naming

All files go in `docs/` directory:
```
docs/
  screenshot-main.png
  screenshot-discord.png
  screenshot-prediction.png
  screenshot-indicators.png
  screenshot-discord-message.png
```
